# simple-slide テンプレート

新しいスライドを作るときのコピー元テンプレートです。

## 使い方

1. このフォルダ(`templates/simple-slide/`)を `YYYY/MM/slide-name/` にコピーする
   - 例: `2026/06/my-presentation/`
   - フォルダ名は半角英数字とハイフンのみ(例: `school-report`, `plan-share`)
2. コピーした `index.html` 内の `★ここを書き換える` コメント部分を書き換える
   - タイトル、日付、パンくず(年・月)、本文
3. 月別一覧ページ(`YYYY/MM/index.html`)にリンクを追加する
4. このREADME.mdはコピー先では削除するか、スライドの説明に書き換える

## 注意

- `index.html` の `<link>` は2行ありますが、テンプレート内プレビュー用と
  コピー先用の両方に対応するためのものです。コピー後は1行目を削除して構いません。
- パンくずと「戻る」リンクはコピー先(`YYYY/MM/slide-name/`)を基準にしているため、
  `templates/` 内でプレビューしている間は正しく機能しません(仕様です)。
- 外部ライブラリは使わず、静的HTML/CSS/JSのみで完結させてください。
