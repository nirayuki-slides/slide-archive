# CLAUDE.md

このリポジトリは、GitHub Pagesで公開する共有用HTMLスライドのアーカイブです。

- 公開URL: https://nirayuki-slides.github.io/slide-archive/
- 詳しい運用ルールは README.md を参照

## Claude Codeが作業するときのルール

### 技術構成

- 静的HTML/CSS/JSのみで構成する
- React・Vite・Next.js・npm・CDN含む外部ライブラリを勝手に追加しない
- ビルド工程を必要とする仕組みを導入しない
- 文字コードはUTF-8、`lang="ja"` の日本語ページを前提にする

### スライドの追加・変更

- 既存スライドを勝手に削除・改変しない(明示的に指示された場合のみ)
- 新規スライドは `YYYY/MM/slide-name/index.html` の形で追加する
  - フォルダ名は半角英小文字・数字・ハイフンのみ
- スライドを追加したら、月別一覧 `YYYY/MM/index.html` へのリンク追加を必ず行う
  - 新しい月を作ったら年別一覧 `YYYY/index.html` にも追加
- リンクはすべて相対パスで書く
  - ローカル絶対パス(`B:\` など)や `file:///` で始まるパスを残さない
  - 参照名と実ファイル名の大文字小文字を一致させる(GitHub Pagesは大文字小文字を区別する)

### 素材・コンテンツ

- 画像や動画など重い素材を大量に入れない
- 画像や動画が多いスライドを追加するときは、ファイルサイズを確認する
- **画像圧縮はしない方針**: スライドの見た目を優先するため、現時点では画像圧縮を行わない。
  既存画像の圧縮・WebP/JPEG変換・再保存を勝手に実行しない。
  表示速度が問題になった場合のみ、ユーザーの指示を受けて別途圧縮を検討する。
- GitHub Pages公開前提なので、個人情報・内部資料・非公開情報を入れない
  (archive/ フォルダも公開されるため同様。ファイル名に個人名を入れるのも不可)
- `_local/` はGit管理外のローカル作業置き場。PPTX原本、生成プロンプト、作業メモ、バックアップなど公開しない手元用ファイルだけを置く

### 不明な点があるとき

- 削除・大きな構成変更が必要に見える場合は、実行前にユーザーに確認する
